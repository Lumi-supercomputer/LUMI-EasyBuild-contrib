# SciPy-bundle license information
-   NumPy is covered by a permissive BSD 3-Clause license, a copy of which can be found in the
    [LICENSE.txt file in the NumPy GitHub repository](https://github.com/numpy/numpy/blob/main/LICENSE.txt).

-   SciPy is also covered by a permissive BSD 3-Clause license, a copy of which can be found in the
    [LICENSE.txt file in the SciPy GitHub repository](https://github.com/scipy/scipy/blob/main/LICENSE.txt).

After installing and loading the module, the license information for most packages in 
the bundle can also be found in
`$EBROOTSCIPYMINBUNDLE/share/licenses`.
