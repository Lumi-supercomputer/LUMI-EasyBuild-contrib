# SuperLU_DIST license information

The SuperLU license can be found in the
[License.txt filel on the SuperLU web site](https://portal.nersc.gov/project/sparse/superlu/License.txt)
or the
[License.txt file in the GitHub repository](https://github.com/xiaoyeli/superlu_mt/blob/master/License.txt).

After installing and loading the module, this information is also available in 
`$EBROOTSUPERLU_DIST/share/licenses/SuperLU_DIST'.
