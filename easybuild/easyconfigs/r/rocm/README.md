# ROCm

### 6.1.3 and 7.0.3

-   As previous ROCm Easyconfigs, but with support for the address sanitizer and 
    debug symbols also. However, the libraries for the address sanitizer and 
    debug symbols need to be activated with `LD_PRELOAD`.

-   Documentation:

        -   [ROCm 6.1.3 documentation (6.1.2 as that is the closest available)](https://rocm.docs.amd.com/en/docs-6.1.2/)
        -   [ROCm 7.0.3 documentation (7.0.2 as this is the closest available)](https://rocm.docs.amd.com/en/docs-7.0.3/)
